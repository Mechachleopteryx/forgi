forgi.config module¶

Code for reading and writing the forgi configuration files.

To configure forgi, run forgi_config.py

forgi.config.iter_configfiles()[source]¶

Iterate over config file locations, from lowest priority to highest priority. The file does not have to exist.

forgi.config.read_config()[source]¶

Read all configuration files and return a configuration dictionary

Returns:A dictionary.
